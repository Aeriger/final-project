There are several different types of animation. A few general ones are explained below, but they all boil down to the basic premise of using multiple images to create an illusion of movement (as seen below).



Traditional Animation: As the name implies, traditional animation involves frame that are individually hand drawn; this method was the most used during the 20th century. Companies like Disney and Studio Ghibli still often rely on such methods to create their films as opposed to using modern software.

Stop Motion Animation: This is animation utilizes the manipulation of physical objects, taking several images in order to create movement. Popular uses of stop motion include clay animation and puppet animation. Great films like Coraline and Chicken Run both use stop motion animation.

Computer Animation: Easily the most modern type of animation, computer animation can actually be used to achieve both 2D and 3D works. 2D works can be made without the meticulousness of traditional animation, while 3D works can be made using modeling software to create nearly photorealistic results.
